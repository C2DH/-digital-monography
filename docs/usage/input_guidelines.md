# Input guidelines

## Package structure requirements

As an author, you are required to upload a package of a following structure:

```
.
├── config.toml
├── part1.md
├── part2-or-some-other-name.md
├── img/
│   ├── part1/
│   │   ├── img1.jpg
│   │   ├── img2.png
│   │   ├── img3.gif
│   │   └── some-img-name.tiff
│   └── part2-or-some-other-name/
│       └── img1.jpg
├── video/
│   └── part2-or-some-other-name/
│       └── video-name.mp4
└── csv/
    └── part1/
        └── some-data.csv
```
